# Catppuccin Logo Wallpapers

The wallpapers in this directory were created using the below command using imagemagick, the command below is a example for catppuccin-logo-mocha-base.png.

```bash
convert -resize 256x256 -extent 2560x1440 -gravity center -background '#1e1e2e' catppuccin-logo.png catppuccin-logo-mocha-base.png
```
